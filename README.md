This is a series of arduino IDE sketches for used on the Waveshare ESP32-S3 1.69inch display development board.

The initial release is a standalone display which uses the internal IMU to draw an artificial horizon.  There are currently four AHI designs, which can be scrolled through by using the PWR button on the Waveshare board.

Simply supply 5v to the grey wire and Ground to the purple wire on the pinout cable (or power via USB).

Portrait and landscape versions are available.
